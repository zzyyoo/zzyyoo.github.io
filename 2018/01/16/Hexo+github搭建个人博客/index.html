#前言
搭建博客的过程中，遇到很多问题，为了让你更快的搭建博客，我来教大家怎么搭建。

##准备工作

  1. 安装Node.js官网下载，用于生成静态页面，下一步下一步..
  2. 安装Git官网下载，国内下载很慢，需要科学上网。
  3. 注册GitHub账号官网
    ##安装hexo
    准备工作做好后，就要安装hexo了,打开Git Bash,输入命令
    npm install hexo-cli -g

    ##初始化
    先在电脑创建一个文件夹，如：blog，然后输入
    cd D:\blog
    ,注意要写成你自己文件夹的路径，然后输入
    hexo init，等他初始化完成，完成后blog文件夹就会有hexo文件了

    ##生成静态页面
    前面只是准备工作，现在才是开始，继续输入hexo generate或者hexo g，生成静态页面。

    ##启动本地服务
    继续输入hexo server或者hexo s来启动服务,
    接着在浏览器输入http://localhost:4000，如果显示下图，那么恭喜你配置成功。
